Executive Summary

The purpose of this Request for Proposal (RFP) is to develop an upgrade kit for off-the-shelf ride-on toys designed specifically for special needs children...

Current State and Use Cases

Currently, off-the-shelf ride-on cars have several limitations that affect their usability for special needs children...

Critical Requirements

  • Steering Lockout: Allows the remote driver to prevent the passenger from providing any steering input...

Optional Features

  • Hot-Swappable or Upgraded Batteries: Allows for extended usage without frequent recharging...
